# Trace — User Flow

## Core User Journey

### 1. Open the app (Dashboard / Landing Page)
- See dashboard with multiple sections:
  - **Map view** — world map with pins showing where you've been
  - **Trip list** — organized trips (past, current, upcoming)
  - **Stats** — countries visited, photos taken, trips completed
  - **Quick actions** — create new trip, add memory, pin a location

### 2. Map View (optional)
- See world map with pins for all visited locations
- Click a pin to see photos, memories, documents from that place
- Can be as simple as "Dubai Airport, June 2026" or as rich as 50 photos + journal
- Filter by date range, trip, or location type

### 3. Trip List (from dashboard)
- See list of trips (past, current, upcoming)
- Each trip shows: name, dates, cover photo, day count

### 4. Select a trip
- See trip overview:
  - Trip name, dates, locations
  - Day-by-day itinerary
  - Photo count, document count
  - Quick stats (days until departure, days remaining, etc.)

### 3. View a specific day
- See all activities for that day
- See photos taken that day
- See documents for that day (boarding passes, tickets)
- Add new activity, photo, or document

### 4. Add an activity
- Name, time, location, notes
- Optional: link to booking confirmation
- Optional: attach document

### 5. Upload a photo
- Select from device or take new
- Add caption/note
- Auto-tag with date, location (if available)
- Optional: add to specific activity

### 6. Store a document
- Upload PDF, image, or other file
- Tag with trip, day, activity
- Add notes (e.g., "Flight confirmation," "Hotel booking")

### 7. Search
- Search by trip name, activity name, photo caption, document tag
- Filter by date range, location, trip

## Key Screens

1. **Dashboard** — landing page, home base with map, trip list, stats, quick actions
2. **Map View** — world map with pins, click to see location details
3. **Trip List** — all trips (accessible from dashboard)
4. **Trip Overview** — details for one trip
5. **Day View** — activities, photos, documents for one day
6. **Activity Detail** — single activity with notes, documents
7. **Photo Gallery** — all photos for a trip, filterable
8. **Document Library** — all documents for a trip, filterable
9. **Travel Log** — list of all cities/countries visited
10. **Search** — global search across all data

## User Personas

### Trent (Primary User)
- Travels frequently (cruise ships, international trips)
- Takes lots of photos
- Needs to keep boarding passes, tickets, confirmations organized
- Currently uses Word documents (painful)
- Wants one place for everything

### Future Users
- Other travelers who want to organize trips
- Couples/families traveling together
- Business travelers with lots of documents
- Anyone who wants to keep travel memories organized
